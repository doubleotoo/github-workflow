A Ruby wrapper to automate a GitHub workflow (using the GitHub REST API v3).


## Installation

Install the gem by issuing

```ruby
gem install github_flow
```

or put it in your Gemfile and run `bundle install`</tt>

```ruby
gem "github_flow"
```
